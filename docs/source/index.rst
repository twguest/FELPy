Welcome to FELpy's documentation!
===================================

**FELpy** is a Python library that extends SRW/WPG and offers a *simple* and *intuitive* API for beamline design and virtual diagnostics.
FELpy has its documentation hosted on Read the Docs.
Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
